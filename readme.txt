
spousteni z prikazove radky:

1. priprava
    cd eld-dip
    bundle install
2. spusteni skriptu
    ruby script/<script>.rb
kde <script> je
    create_db - vytvori databazi database.sqlite pro ostatni skripty, predoklada existenci souboru s daty:
            data/matice.csv
            data/obce.dbf
            data/RGR_regiony_rozdeleni.xlsx
            data/Tik_Tki_Tii.xls
    export
    jadra
